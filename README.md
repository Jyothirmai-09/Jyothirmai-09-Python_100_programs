# 🐍 Python 100 Programs

A collection of **100 Python programs** covering fundamental programming concepts, problem-solving techniques, strings, lists, dictionaries, searching, sorting, number systems, pattern printing, and basic data structures.

This repository is designed for **beginners, students, lab practicals, coding practice, and interview preparation**.

* ✅ 100 beginner-friendly Python programs

* ✅ Pattern printing programs

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/Python_100_Programs.git
```

2. Navigate to the project folder:

```bash
cd Python_100_Programs
```

3. Run any program:

```bash
python 01_palindrome.py
```
